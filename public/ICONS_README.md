# PWA Icons

The icon files (icon-192x192.svg and icon-512x512.svg) are placeholder SVG files.

To replace them with proper PNG icons:
1. Create or design your app icon
2. Export as PNG at 192x192 and 512x512 sizes
3. Replace the .svg files with .png files
4. Update manifest.json to reference .png files

You can use tools like:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/
